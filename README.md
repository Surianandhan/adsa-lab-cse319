# Analysis and Design of Algorithms (ADSA)

This repository contains implementations of fundamental algorithms covering sorting, dynamic programming, greedy methods, graph algorithms, and backtracking.

## Topics Covered

### Sorting Algorithms
- Insertion Sort
- Merge Sort

### Brute Force
- Travelling Salesman Problem (TSP - Permutation)

### Dynamic Programming
- Rod Cutting (Recursive, Memoization, Bottom-Up)
- Matrix Chain Multiplication
- Optimal Binary Search Tree (OBST)
- 0/1 Knapsack

### Greedy Algorithms
- Fractional Knapsack
- Job Sequencing with Deadlines

### Backtracking
- N-Queens Problem
- Sum of Subsets

### Branch and Bound
- Travelling Salesman Problem
- 0/1 Knapsack

### Graph Algorithms
- Topological Sort
- Prim’s Algorithm
- Kruskal’s Algorithm
- Bellman-Ford Algorithm
- Floyd-Warshall Algorithm

### Network Flow
- Ford-Fulkerson Algorithm

## How to Run

Compile:
g++ filename.cpp -o output

Run:
./output

## Key Learnings

- Understanding time complexity differences between algorithms
- Applying dynamic programming for optimization problems
- Using greedy strategies effectively
- Handling NP-hard problems using approximation or backtracking
- Working with graph-based algorithms and shortest paths
